### (WIP) Git GUI client written in Dart and uses Flutter.
---

Preffered flutter version: `stable, 2.5.0`

#### Get started

1. `git@github.com:ivk1800/diff.git`
2. `cd diff/tools`
3. `./packages_get.sh`
4. `./gen.sh`
5. `cd ../launch`
6. `flutter run --release`
